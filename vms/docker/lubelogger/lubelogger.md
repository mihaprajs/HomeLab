# LubeLogger Notes

Used the official docker compose file from their [GitHub](https://github.com/hargata/lubelog).

**R Script in this directory is specifically made to convert CSVs exported from ([SpritMonitor.de](https://www.spritmonitor.de/)).
Most likely it won't work with other CSVs.** It was made using ChatGPT and requires R to be installed on your machine.
